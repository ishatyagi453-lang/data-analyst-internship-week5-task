#  Apple Stock Price Analysis Using Python

## 📌 Project Overview

This project analyzes the historical stock prices of **Apple Inc. (AAPL)** using Python. The stock data was collected directly from
**Yahoo Finance** using the `yfinance` library. The analysis focuses on identifying stock price trends, trading activity, daily 
returns, moving averages, and volatility through Exploratory Data Analysis (EDA) and data visualization.


## 🎯 Project Objectives

- Collect Apple stock data using Python.
- Perform Exploratory Data Analysis (EDA).
- Analyze historical stock price trends.
- Calculate Moving Averages (7-Day and 30-Day).
- Measure Daily Returns and Volatility.
- Create meaningful visualizations.
- Generate business insights from the analysis.


## 📂 Dataset Information

- **Source:** Yahoo Finance (`yfinance`)
- **Company:** Apple Inc.
- **Ticker Symbol:** AAPL
- **Time Period:** January 2020 – January 2025

### Dataset Features

| Column | Description |
|---------|-------------|
| Date | Trading Date |
| Open | Opening Stock Price |
| High | Highest Stock Price |
| Low | Lowest Stock Price |
| Close | Closing Stock Price |

## 💡 Key Insights

- Apple stock showed an overall upward trend during the analysis period.
- The 7-Day Moving Average captured short-term price movements.
- The 30-Day Moving Average highlighted the long-term trend.
- Trading volume varied over time, reflecting changes in investor activity.
- Daily returns showed normal market fluctuations.
- Higher volatility was observed during periods of significant price changes.

## ✅ Conclusion

The analysis demonstrates that Apple experienced strong long-term growth despite short-term market fluctuations. 
Using Python, historical stock data was analyzed through EDA, feature engineering, and visualization to understand market behavior 
and identify important financial trends.
